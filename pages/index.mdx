
# Введение

Данная пользовательская документация предназначена для инструктирования по работе с панелью администраторов веб-приложения "Твоя шаурма".

**Твоя шаурма**  - веб-приложение одноименной сети ресторанов.
Предназначено для клиентов, позволяет ознакомится с деятельностью компании, ассортиментом продукции, выбрать товар из каталога и оформить заказ. 

Основное (клиентское) приложение доступно по адресу: [**https://next-junkfood.vercel.app**](https://next-junkfood.vercel.app) 

Панель администраторов **"Твоя шаурма"** представляет собой систему управления контентом основного веб-приложения "Твоя шаурма" и обладает функционалом:
- просмотра статистики (доход ресторана, количество чеков, график продаж)
- изменения, добавления, удаления категорий товаров;
- изменения, добавления, удаления самих товаров (включая название, описание, изображения, стоимость);
- отслеживания новых заказов
- изменения статуса выполнения заказов в части статусов доставки;
- отмены оплаты невыполненных заказов
- и др. 

Панель администраторов **"Твоя шаурма"** доступна по адресу [**https://nuxt-junkfood-admin.vercel.app/**](https://nuxt-junkfood-admin.vercel.app/)
