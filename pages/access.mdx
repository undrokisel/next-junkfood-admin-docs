# Права доступа

В целях безопасности админ-панель размещена на отдельном домене и защищена системой аутентификации. 

Вход в систему осуществляется по коду, высланному на электронную почту, при помощи сервиса аутентификаии [**Clerk**](https://clerk.com/).

Для получения доступа к админ-панели необходимо обратиться к владельцу веб-приложения "Твоя шаурма" 
либо к руководству отдела разработки (по почте **twoya-shaurma@dev.com** либо через телеграм **@twoya_shaurma_dev**)
и указать свой email.

# База данных
В случае крайней необходимости возможен прямой доступ к базе данных и ее рактирование при помощи инструмента [**PrismaStudio**](https://www.prisma.io/studio).
Для получения прямого доступа к базе данных необходимо обратиться в отдел разработки.
