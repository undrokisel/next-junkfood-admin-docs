# Права доступа

В целях безопасности админ-панель размещена 
на отдельном домене и защищена системой аутентификации. 

Доступ предоставляется исключительно уполномоченным сотрудникам по запросу к владельцу ПО. Для активации:
- предоставить корпоративный email отделу разработки;
- подтвердить приглашение в системе Clerk (перейдя по ссылке в письме);
- пройти аутентификацию при первом входе;


# База данных
В случае крайней необходимости возможен прямой доступ к базе данных и ее редактирование при помощи инструмента [**PrismaStudio**](https://www.prisma.io/studio).
Для получения прямого доступа к базе данных необходимо обратиться в отдел разработки.
