# Главная страница панели администратора


На главной странице панели администратора отображена информация о показателях работы ресторана, таких как: 

- доход ресторана (по оплаченным заказам накопительным итогом);
- количество чеков (по оплаченным заказам накопительным итогом);
- количество видов товаров. 

Также на главной странице отображен график объема продаж с разделением по месяцам. 

Наверху каждой страницы панели администратора размещена навигационная панель.
