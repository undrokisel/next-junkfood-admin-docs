# Все пользователи

На странице в виде таблицы отображены все пользователи. 
Отображаются данные: 
- ID пользователя
- имя (и фамилия)
- email
- дата подтверждения аккаунта
- провайдер авторизации
- дата создания аккаунта

![search](/images/users.png)

## Редактирование пользователей
Редактированный данных пользователей через панель администратора не предусмотрено.
