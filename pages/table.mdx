# Работа с таблицами

В разделах "Категории", "Товары", "Заказы", "Пользователи" Вы можете увидеть перечень соответствующих данных в табличном отображении. 

## Общее количество строк

Рядом с заголовком таблицы отображается цифра, которая обозначает количество строк в таблице: 

![search](/images/category1.png)


## Пагинация
В случае, если количество строк в таблице превышает 10, пользователь может воспользоваться пагинацией. Кнопки пагинации расположены внизу таблицы: 

![search](/images/pag.png)

## Поиск
Пользователь может выполнить фильтрацию данных по полю поиска, которое расположено выше левого края таблицы: 

![search](/images/search.png)

## Горизонтальная прокрутка
Если ширина таблицы больше ширины экрана, пользователь может выполнить горизонтальную прокрутку вправо с помощью клавиаутуры или тачпада.


## Действия со строкой
В таблицах могут быть предусмотрены действия со строками. 
Выберите строку, с которой будет совершено действие, прокрутите таблицу вправо, пока не увидите отображение столбца "Действия":

![search](/images/actions.png)

В соответствующей строке нажмите на "...", откроется окно для выбора действий со строкой:  

![search](/images/actions2.png)

## Удаление строки

При нажатии на кнопку "Удалить" откроется диалоговое окно: 

![search](/images/modal.png)

При нажатии на кнопку "Продолжить" строка из таблицы базы данных будет удалена, а браузер выполнит обновление страницы.

## Редактирование строки

При нажатии на кнопку "Редактировать" браузер выполнит переадресацию на страницу редактирования. 

Измените необходимые поля и нажмите "Сохранить изменения":

![search](/images/save.png)

## Создание (добавление новой) строки

При нажатии на кнопку "+ Создать ...", которая находится в правом верхнем углу страницы, браузер выполнит переадресацию на страницу создания строки. 

![search](/images/create2.png)

Заполните необходимые поля и нажмите "Создать":

![search](/images/create.png)

## Пути изображений

При создании и редактировании строк с изображениями необходимо учесть следующее. 

Путь к изображению можно указать двумя способами:
    1. абсолютный адрес в сети Интернет, например https://picme.image1.jpeg, если изображение доступно в глобальной сети.
    2. относительный адрес, если изображение размещено на сервере веб-приложения "Твоя шаурма". 
    
    Во втором случае изображение должно находится в директории /public/images в корне проекта, 
    а указываемый путь должен соответствовать этому шаблону "/imаges/путь_к_изображению.формат_изображения"
